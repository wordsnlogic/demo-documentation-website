# AcmeCloud Release Notes

This document provides a history of all major and minor releases of AcmeCloud.

---

## Version 1.3.0 – 2026-02-01

**Highlights:**

- Introduced advanced reporting dashboards.
- Added team-level permissions for projects.
- Improved API response times by 30%.
- Fixed bug in user invitation workflow.

---

## Version 1.2.0 – 2025-12-15

**Highlights:**

- Added support for webhook notifications.
- Enhanced project templates for faster setup.
- Updated API documentation with new endpoints.
- Fixed UI issues in project overview page.

---

## Version 1.1.0 – 2025-10-10

**Highlights:**

- Introduced analytics and reporting module.
- Added API throttling and rate limiting.
- Improved project creation workflow.
- Minor bug fixes across the platform.

---

## Version 1.0.0 – 2025-08-01

**Highlights:**

- Initial public release of AcmeCloud.
- Core modules: Projects, Users, APIs, Reporting.
- Fully functional REST API for integrations.
- Secure authentication with role-based permissions.

---

**Release Notes Guidelines:**

- Each release includes a version number, release date, and key highlights.
- Minor bug fixes or changes are documented briefly.
- Major new features and breaking changes are clearly highlighted.
