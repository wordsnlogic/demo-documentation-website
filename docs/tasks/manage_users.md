# Manage users

This guide explains how to manage users in AcmeCloud.

## Adding Users

1. Navigate to the **Users** tab.
2. Click **Invite User**.
3. Enter the email address and role.
4. Click **Send Invitation**.

## Managing Permissions

- Roles: Admin, Editor, Viewer.
- Permissions can be changed anytime via the **Edit Role** option.
- For large teams, consider using group-based permissions.
