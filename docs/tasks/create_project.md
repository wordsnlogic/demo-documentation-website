# Create a Project

This guide explains how to create a new project in AcmeCloud.

## Steps

1. Navigate to the **Projects** tab.
2. Click **Create New Project**.
3. Enter the project name, description, and select a template.
4. Assign team members and set roles.
5. Click **Create**.

## Tips

- Use meaningful project names for easy tracking.
- Templates can speed up project setup.
- Always assign a project owner.
